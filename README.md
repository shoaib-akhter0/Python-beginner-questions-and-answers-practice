## 📝 Notebook Format

Each exercise in the notebook follows a simple structure:

### Question

A programming problem is presented.

```python
# Example Question
Write a Python program to check whether a number is even or odd.
```

### Solution

```python
num = int(input("Enter a number: "))

if num % 2 == 0:
    print("Even")
else:
    print("Odd")
```

This format makes it easy to understand both the problem and its solution in one place.
